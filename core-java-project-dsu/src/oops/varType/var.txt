Var Type Name
~~~~~~~~~~~~~

The var reserved type name (not a Java keyword) was introduced in Java 10. 

Type inference is used in var keyword in which it detects automatically 
the datatype of a variable based on the surrounding context.

Usage
~~~~~~~
1. We can declare any datatype with the var keyword.
2. var can be used in a local variable declaration.

Note:
~~~~~~
Cannot use var on variables without initializer
